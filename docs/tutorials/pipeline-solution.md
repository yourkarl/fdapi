### 方案概述

地下综合管线以给排水、燃气、电力、通信等多专业管线及综合管廊的三维模型为底座，
承载管线资产管理、空间分析、运行监测与应急处置，将 GIS 管网数据与 IoT 融合，
依托飞渡 DTS（`fdapi`）构建「透明地下」数字孪生。

---

## 管线三维底座

| 能力点 | fdapi 接口 | 说明 |
|------|-----------|------|
| 管线 / 管廊建模 | `tileLayer` / `polyline` + `getObjectIDs` | 多专业管线、管廊单体化与拾取 |
| 属性查询 | `getActorInfo(FromDB)` / `marker.popupURL` | 管径、材质、埋深、权属、建设年代查询 |
| 地形 / 地表 | `globeTerrain（球面）` / `imageryLayer2（球面）` + `enableClip` | 地表半透与地下管线透视 |
| 专业分层 | `infoTree` / `groupId` | 按给水、排水、燃气、电力、通信分色分组 |

## 空间分析

| 能力点 | fdapi 接口 | 说明 |
|------|-----------|------|
| 横 / 纵断面 | `editHelper` + `polyline` | 沿线开挖横断面、纵断面查看管线关系 |
| 埋深 / 净距分析 | `coord` / `tools` | 埋深量算、管线间距与碰撞检查 |
| 区域圈选统计 | `editHelper` 画范围 + 点在面内判断 | 框选区域统计管线里程 / 数量并高亮 |
| 缓冲分析 | `tools` / `queryOption` | 管线影响范围缓冲查询 |

## 运行监测与应急

| 能力点 | fdapi 接口 | 说明 |
|------|-----------|------|
| 监测点位 | `marker` / `customTag` + `update` | 流量、压力、液位、井盖等测点状态 |
| 报警定位 | `marker.focus` + `highlightArea` | 爆管、渗漏、超压报警联动定位高亮 |
| 爆管 / 内涝分析 | `floodFill` / `highlightArea` | 爆管影响范围、内涝淹没模拟与影响管段 |
| 巡检养护 | `polyline` + `marker` | 巡检轨迹、缺陷点位与养护工单联动 |

---

## 要点

- 地下管线量大且线状，海量管段用 `groupId` 分专业分组显隐 + 批量提交渲染；
- 地表半透 + `enableClip` 实现「透明地下」，突出目标管线；
- 埋深、净距、断面分析依赖 `coord` / `editHelper` / `tools`，坐标需与 GIS 管网统一；
- 爆管、内涝等分析涉及 `floodFill` 等授权分析类，详见「高级接口授权」。
