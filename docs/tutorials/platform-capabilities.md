# 平台技术优势与性能指标


### 概述

飞渡 DTS 是以「CIM 数据引擎 + 云渲染 + 高逼真可视化」为核心的国产自主数字孪生平台，产品矩阵：

- **CIM 数据引擎（iFreedo 飞渡芯）**：BIM+GIS 数据无损转换、多源融合与轻量化，大体量 BIM「秒级体验」。
- **轻量化可视化平台（HoloLen3D For Web）**：基于 WebGL 的无插件、跨系统 / 浏览器三维应用平台，前端 JS 接口 + 后端 RESTful，支持 .NET / Java 等主流语言。
- **高逼真可视化平台（Holo3DScene）**：BIM + 倾斜 + 影像 + 点云 + MAX 大场景高逼真渲染，L1–L5 照片级精度输出。

---

## 核心技术优势

### 1. 数据轻量化

- **格式转换**：多源数据统一为通用标准格式，支持地形影像、倾斜摄影、MAX、BIM、点云、矢量、UE 等；转换即轻量化，输出瓦片 + 金字塔 + 3D 模型 LOD 金字塔。
- **数模分离**：BIM 几何数据与属性 / 结构数据分离，属性可存 XML 或入 PG 库并保留与图形的唯一关联（图属分离）。
- **轻量优化**：多重 LOD、三角面化简、纹理压缩（Webp / CRN / KTX2）、Draco 压缩，PBR 材质优化；让渲染性能与数据规模 / 复杂度无关。
- **超大体量**：支持 100G 以上、千万级构件的 BIM 轻量化；优化后十亿级三角面、千万级构件流畅加载。
- 轻量化后可见结构无丢失、纹理无明显下降、LOD 无明显失真；Web / 大屏端加载帧率 ≥ 25fps。

### 2. 云渲染

- **服务端实时渲染 + 视频串流**：渲染画面实时推流到终端，无插件、跨平台、跨浏览器，操作双向实时反馈。
- **分布式部署**：多节点分布式渲染，主节点控制子节点资源与数据同步；实例工程配置、访问权限、IP 白名单（详见 [实例管理接口](/instance-manager) 与 [云端部署](/docs/tutorials/cloud-deploy)）。
- **多端适配**：电脑 / 平板 / 手机 / 大屏，微信或浏览器访问。

### 3. 海量数据 LOD 优化与高逼真渲染

- **多层级 LOD 切片**（源自 WMTS 瓦片理念）：动态轮廓模型与精细实体模型的内存加载与渲染，解决城市级海量数据性能瓶颈。
- **高逼真渲染引擎**：室内外实时光照与阴影、PBR 物理材质、动态水 / 海水、夜景与昼夜交替、粒子、后期特效（线框 / LUT / 科幻风）；L1 城市级 → L5 构件级精度。

### 4. CIM 数据底板与二三维一体化

- 横向融合城市全生命周期的 BIM / GIS / IoT，纵向贯穿各业务系统，构建 CIM 数据底板。
- 几何定义符合 **OGC / IFC / I3S** 等标准，可与 ArcGIS / Revit / MicroStation / CATIA / PDMS 等主流 BIM/GIS 软件数据交换。

### 5. 微服务与二次开发

- 平台功能以 **RestAPI 微服务**开放（量测、剖切、标绘、道路绘制、挖洞、路径动画、部件 / 模型选取、模型编辑等），经授权可与外部系统集成。
- **700+ 二次开发接口**、**100+ 开发样例**，提供 SDK 包；支持 C/S 与 B/S（B/S 多前端框架），前端 JS + 后端 RESTful。

## 关键性能指标

| 指标 | 要求 |
|------|------|
| 二维瓦片 / 动态矢量服务 | 500 并发 · 亿级数据 · 响应 ≤ 500ms；5000 并发 · 亿级数据 · ≤ 2000ms |
| 三维瓦片服务 | 初始加载 ≤ 1000ms，高精度显示 ≤ 2000ms，大范围（≥1000km²）渲染 ≤ 2000ms |
| 全要素调度 | 不限可视距离下任意视角全部可见数据调度 ≤ 800ms |
| 倾斜摄影 | 5000km² 流畅加载显示，帧率 ≥ 25fps |
| 城市级 MAX 建筑 | 5000km² 以上流畅加载，帧率 ≥ 25fps |
| 高分地形影像 | 十万 km² 以上流畅加载，帧率 > 25fps |
| 管网数据 | 5000km² 以上流畅加载，帧率 > 25fps |
| 矢量数据 | 千万级对象流畅加载，帧率 ≥ 25fps |
| BIM 模型 | 帧率 ≥ 25fps，响应 ≤ 700ms |
| 查询响应 | 基础查询 ≤ 300ms，复杂查询 ≤ 500ms，空间信息查询 ≤ 2000ms |

## 数据格式支持（轻量化 / 转换）

| 类别 | 格式 |
|------|------|
| 栅格（影像 / 地形） | tif、img、jpg |
| 倾斜摄影 | osgb |
| 手工模型 | max、fbx、xml+osg、datasmith、3ds、obj、dae、stl |
| BIM | rvt、dgn、ifc、CATProduct、3DXML、nwd、SolidWorks、skp、stp/step |
| 点云 | las / laz |
| 矢量 | shp（可发布为三维场景） |
| 输出标准格式 | 3DTiles、I3S、S3M、osgb |

> 兼容 OGC、IFC、ISO15926、STEP AP203/AP214 等标准；支持无需预处理的 OGC I3S↔3DTiles 实时在线转换。

## 国产化与安全

- **国产化适配**：国产操作系统（银河麒麟 / 中标麒麟 / 统信 UOS / 华为欧拉）、国产数据库（达梦 / 瀚高 / 人大金仓 / GaussDB / PolarDB 等）、国产 CPU（鲲鹏 / 海光 / 飞腾 / 申威 / 兆芯）、国产中间件；自主知识产权。
- **安全**：用户权限与访问控制、身份认证与会话（密码复杂度 / 加密 / 自动登出）、关键数据加密、数据备份、分级分类日志（运维 / 异常 / 告警 / 登录）。
