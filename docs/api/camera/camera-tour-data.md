# CameraTourData

相机导览数据对象

## 构造函数

```js
new CameraTourData(id, name, keyFrames)
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `id` | `string` | 字符串类型的ID |
| `name` | `string` | 名称 |
| `keyFrames` | `array` | 关键帧对象数组 [对象结构示例](/docs/api/camera/camera-tour-key-frame) |
