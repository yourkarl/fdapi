# QueryOption

数据库查询选项

## 构造函数

```js
new QueryOption()
```

## 成员（Members）

| 成员 | 类型 | 说明 |
|------|------|------|
| `fields` | — | 获取要查询的字段（列） |
| `fields` | — | 设置要查询的字段（列） |
| `id` | — | 获取TileLayer的ID |
| `id` | — | 设置TileLayer的ID |
| `limit` | — | 用于数据库分页查询，限制取多少条数据（即每页显示多少条记录） |
| `offset` | — | 用于数据库分页查询，跳过多少条数据然后取后续数据 |
| `where` | — | 获取要查询的条件 |
| `where` | — | 设置要查询的条件 |
